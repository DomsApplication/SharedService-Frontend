https://www.youtube.com/watch?v=fzxEECHnsvU&t=3390s

1. create react app
npx create-react-app <app-name>

2. install react router
npm install react-router-dom@6

3. intsall Google font 'Nunito' plugin
step-1: https://fonts.google.com/specimen/Nunito?query=nunito
step-2: generate the below link and placed into public/index.html
<link href="https://fonts.googleapis.com/css2?family=Nunito:ital,wght@0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;0,1000;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900;1,1000&display=swap" rel="stylesheet">

4. Copy the css from generated google font 'Nunito' plugin from the url.
 https://fonts.google.com/specimen/Nunito?query=nunito
then paste the below style into public/index.html
  <style>
    * {
      font-family: 'Nunito', sans-serif;
      margin: 0;
    }
  </style>


5. install the sass style sheet   
npm install sass

6. install material UI from the mui.com
npm install @mui/material @emotion/react @emotion/styled

7. install material UI  FONT from the mui.com
npm install @mui/icons-material

8. install material UI  DATA-GRID from the mui.com
npm install @mui/x-data-grid

9. cleanup the public folder, remove all the files except 'index.html'

10. cleanup src folder, remove all the files except 'index.js' & 'App.js'
